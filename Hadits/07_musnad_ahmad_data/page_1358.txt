حَدَّثَنَا سُفْيَانُ قَالَ عَمْرٌو وَسَمِعْتُ عِكْرِمَةَ
{ وَإِذْ صَرَفْنَا إِلَيْكَ }
وَقُرِئَ عَلَى سُفْيَانَ عَنِ الزُّبَيْرِ
{ نَفَرًا مِنْ الْجِنِّ يَسْتَمِعُونَ الْقُرْآنَ }
قَالَ بِنَخْلَةَ وَرَسُولُ اللَّهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ يُصَلِّي الْعِشَاءَ الْآخِرَةَ
{ كَادُوا يَكُونُونَ عَلَيْهِ لِبَدًا }
قَالَ سُفْيَانُ كَانَ بَعْضُهُمْ عَلَى بَعْضٍ كَاللِّبَدِ بَعْضُهُ عَلَى بَعْضٍ
