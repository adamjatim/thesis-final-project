Ayat:
حَدَّثَنِي مُحَمَّدُ بْنُ عَبْدِ اللَّهِ حَدَّثَنَا مُحَمَّدُ بْنُ عُبَيْدٍ الطَّنَافِسِيُّ عَنْ الْعَوَّامِ قَالَ سَأَلْتُ مُجَاهِدًا عَنْ سَجْدَةٍ فِي ص فَقَالَ
سَأَلْتُ ابْنَ عَبَّاسٍ مِنْ أَيْنَ سَجَدْتَ فَقَالَ أَوَ مَا تَقْرَأُ
{ وَمِنْ ذُرِّيَّتِهِ دَاوُدَ وَسُلَيْمَانَ }
{ أُولَئِكَ الَّذِينَ هَدَى اللَّهُ فَبِهُدَاهُمْ اقْتَدِهْ }
فَكَانَ دَاوُدُ مِمَّنْ أُمِرَ نَبِيُّكُمْ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ أَنْ يَقْتَدِيَ بِهِ فَسَجَدَهَا دَاوُدُ عَلَيْهِ السَّلَام فَسَجَدَهَا رَسُولُ اللَّهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ
{ عُجَابٌ }
عَجِيبٌ الْقِطُّ الصَّحِيفَةُ هُوَ هَا هُنَا صَحِيفَةُ الْحِسَابِ وَقَالَ مُجَاهِدٌ
{ فِي عِزَّةٍ }
مُعَازِّينَ
{ الْمِلَّةِ الْآخِرَةِ }
مِلَّةُ قُرَيْشٍ الْاخْتِلَاقُ الْكَذِبُ الْأَسْبَابُ طُرُقُ السَّمَاءِ فِي أَبْوَابِهَا قَوْلُهُ
{ جُنْدٌ مَا هُنَالِكَ مَهْزُومٌ }
يَعْنِي قُرَيْشًا
{ أُولَئِكَ الْأَحْزَابُ }
الْقُرُونُ الْمَاضِيَةُ
{ فَوَاقٍ }
رُجُوعٍ
{ قِطَّنَا }
عَذَابَنَا
{ اتَّخَذْنَاهُمْ سُخْرِيًّا }
أَحَطْنَا بِهِمْ
{ أَتْرَابٌ }
أَمْثَالٌ وَقَالَ ابْنُ عَبَّاسٍ الْأَيْدُ الْقُوَّةُ فِي الْعِبَادَةِ الْأَبْصَارُ الْبَصَرُ فِي أَمْرِ اللَّهِ
{ حُبَّ الْخَيْرِ عَنْ ذِكْرِ رَبِّي }
مِنْ ذِكْرِ طَفِقَ مَسْحًا يَمْسَحُ أَعْرَافَ الْخَيْلِ وَعَرَاقِيبَهَا
{ الْأَصْفَادِ }
الْوَثَاقِ
