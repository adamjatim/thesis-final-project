حَدَّثَنَا أَبُو نُعَيْمٍ حَدَّثَنَا زُهَيْرٌ عَنْ أَبِي إِسْحَاقَ أَنَّهُ سَمِعَ رَجُلًا سَأَلَ الْأَسْوَدَ
{ فَهَلْ مِنْ مُدَّكِرٍ }
أَوْ مُذَّكِرٍ فَقَالَ سَمِعْتُ عَبْدَ اللَّهِ

{ فَهَلْ مِنْ مُدَّكِرٍ }
قَالَ وَسَمِعْتُ النَّبِيَّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ يَقْرَؤُهَا
{ فَهَلْ مِنْ مُدَّكِرٍ }
دَالًا
