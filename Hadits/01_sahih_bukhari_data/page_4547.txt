Ayat:
حَدَّثَنَا عُبَيْدُ اللَّهِ بْنُ مُوسَى عَنْ إِسْرَائِيلَ عَنْ مُوسَى بْنِ أَبِي عَائِشَةَ أَنَّهُ سَأَلَ سَعِيدَ بْنَ جُبَيْرٍ
عَنْ قَوْلِهِ تَعَالَى
{ لَا تُحَرِّكْ بِهِ لِسَانَكَ }
قَالَ وَقَالَ ابْنُ عَبَّاسٍ كَانَ يُحَرِّكُ شَفَتَيْهِ إِذَا أُنْزِلَ عَلَيْهِ فَقِيلَ لَهُ
{ لَا تُحَرِّكْ بِهِ لِسَانَكَ }
يَخْشَى أَنْ يَنْفَلِتَ مِنْهُ
{ إِنَّ عَلَيْنَا جَمْعَهُ وَقُرْآنَهُ }
أَنْ نَجْمَعَهُ فِي صَدْرِكَ وَقُرْآنَهُ أَنْ تَقْرَأَهُ
{ فَإِذَا قَرَأْنَاهُ }
يَقُولُ أُنْزِلَ عَلَيْهِ
{ فَاتَّبِعْ قُرْآنَهُ ثُمَّ إِنَّ عَلَيْنَا بَيَانَهُ }
أَنْ نُبَيِّنَهُ عَلَى لِسَانِكَ
