تَلَا
{ إِلَّا الْمُسْتَضْعَفِينَ مِنْ الرِّجَالِ وَالنِّسَاءِ وَالْوِلْدَانِ }
قَالَ كُنْتُ أَنَا وَأُمِّي مِمَّنْ عَذَرَ اللَّهُ
وَيُذْكَرُ عَنْ ابْنِ عَبَّاسٍ
{ حَصِرَتْ }
ضَاقَتْ
{ تَلْوُوا }
أَلْسِنَتَكُمْ بِالشَّهَادَةِ وَقَالَ غَيْرُهُ الْمُرَاغَمُ الْمُهَاجَرُ رَاغَمْتُ هَاجَرْتُ قَوْمِي
{ مَوْقُوتًا }
