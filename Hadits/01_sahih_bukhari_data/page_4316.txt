Ayat:
حَدَّثَنَا أَبُو الْيَمَانِ أَخْبَرَنَا شُعَيْبٌ حَدَّثَنَا أَبُو الزِّنَادِ عَنْ الْأَعْرَجِ عَنْ أَبِي هُرَيْرَةَ رَضِيَ اللَّهُ عَنْهُ
أَنَّ رَسُولَ اللَّهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ قَالَ قَالَ اللَّهُ عَزَّ وَجَلَّ أَنْفِقْ أُنْفِقْ عَلَيْكَ وَقَالَ يَدُ اللَّهِ مَلْأَى لَا تَغِيضُهَا نَفَقَةٌ سَحَّاءُ اللَّيْلَ وَالنَّهَارَ وَقَالَ أَرَأَيْتُمْ مَا أَنْفَقَ مُنْذُ خَلَقَ السَّمَاءَ وَالْأَرْضَ فَإِنَّهُ لَمْ يَغِضْ مَا فِي يَدِهِ وَكَانَ عَرْشُهُ عَلَى الْمَاءِ وَبِيَدِهِ الْمِيزَانُ يَخْفِضُ وَيَرْفَعُ
{ اعْتَرَاكَ }
افْتَعَلَكَ مِنْ عَرَوْتُهُ أَيْ أَصَبْتُهُ وَمِنْهُ يَعْرُوهُ وَاعْتَرَانِي
{ آخِذٌ بِنَاصِيَتِهَا }
أَيْ فِي مِلْكِهِ وَسُلْطَانِهِ عَنِيدٌ وَعَنُودٌ وَعَانِدٌ وَاحِدٌ هُوَ تَأْكِيدُ التَّجَبُّرِ
{ اسْتَعْمَرَكُمْ }
جَعَلَكُمْ عُمَّارًا أَعْمَرْتُهُ الدَّارَ فَهِيَ عُمْرَى جَعَلْتُهَا لَهُ
{ نَكِرَهُمْ }
وَأَنْكَرَهُمْ وَاسْتَنْكَرَهُمْ وَاحِدٌ
{ حَمِيدٌ مَجِيدٌ }
كَأَنَّهُ فَعِيلٌ مِنْ مَاجِدٍ مَحْمُودٌ مِنْ حَمِدَ سِجِّيلٌ الشَّدِيدُ الْكَبِيرُ سِجِّيلٌ وَسِجِّينٌ وَاللَّامُ وَالنُّونُ أُخْتَانِ وَقَالَ تَمِيمُ بْنُ مُقْبِلٍ وَرَجْلَةٍ يَضْرِبُونَ الْبَيْضَ ضَاحِيَةً
ضَرْبًا تَوَاصَى بِهِ الْأَبْطَالُ سِجِّينَا
