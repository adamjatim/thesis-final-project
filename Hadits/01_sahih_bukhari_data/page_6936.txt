Ayat:
حَدَّثَنَا مُسَدَّدٌ عَنْ هُشَيْمٍ عَنْ أَبِي بِشْرٍ عَنْ سَعِيدِ بْنِ جُبَيْرٍ عَنْ ابْنِ عَبَّاسٍ رَضِيَ اللَّهُ عَنْهُمَا
{ وَلَا تَجْهَرْ بِصَلَاتِكَ وَلَا تُخَافِتْ بِهَا }
قَالَ أُنْزِلَتْ وَرَسُولُ اللَّهِ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ مُتَوَارٍ بِمَكَّةَ فَكَانَ إِذَا رَفَعَ صَوْتَهُ سَمِعَ الْمُشْرِكُونَ فَسَبُّوا الْقُرْآنَ وَمَنْ أَنْزَلَهُ وَمَنْ جَاءَ بِهِ فَقَالَ اللَّهُ تَعَالَى
{ وَلَا تَجْهَرْ بِصَلَاتِكَ وَلَا تُخَافِتْ بِهَا }
{ لَا تَجْهَرْ بِصَلَاتِكَ }
حَتَّى يَسْمَعَ الْمُشْرِكُونَ
{ وَلَا تُخَافِتْ بِهَا }
عَنْ أَصْحَابِكَ فَلَا تُسْمِعُهُمْ
{ وَابْتَغِ بَيْنَ ذَلِكَ سَبِيلًا }
أَسْمِعْهُمْ وَلَا تَجْهَرْ حَتَّى يَأْخُذُوا عَنْكَ الْقُرْآنَ
