Nombor hadis: (4138)

Ayat:
حَدَّثَنَا الْحُمَيْدِيُّ حَدَّثَنَا سُفْيَانُ حَدَّثَنَا عَمْرٌو قَالَ سَمِعْتُ مُجَاهِدًا قَالَ سَمِعْتُ ابْنَ عَبَّاسٍ رَضِيَ اللَّهُ عَنْهُمَا يَقُولُ
كَانَ فِي بَنِي إِسْرَائِيلَ الْقِصَاصُ وَلَمْ تَكُنْ فِيهِمْ الدِّيَةُ فَقَالَ اللَّهُ تَعَالَى لِهَذِهِ الْأُمَّةِ
{ كُتِبَ عَلَيْكُمْ الْقِصَاصُ فِي الْقَتْلَى الْحُرُّ بِالْحُرِّ وَالْعَبْدُ بِالْعَبْدِ وَالْأُنْثَى بِالْأُنْثَى فَمَنْ عُفِيَ لَهُ مِنْ أَخِيهِ شَيْءٌ }
فَالْعَفْوُ أَنْ يَقْبَلَ الدِّيَةَ فِي الْعَمْدِ
{ فَاتِّبَاعٌ بِالْمَعْرُوفِ وَأَدَاءٌ إِلَيْهِ بِإِحْسَانٍ }
يَتَّبِعُ بِالْمَعْرُوفِ وَيُؤَدِّي بِإِحْسَانٍ
{ ذَلِكَ تَخْفِيفٌ مِنْ رَبِّكُمْ وَرَحْمَةٌ }
مِمَّا كُتِبَ عَلَى مَنْ كَانَ قَبْلَكُمْ
{ فَمَنْ اعْتَدَى بَعْدَ ذَلِكَ فَلَهُ عَذَابٌ أَلِيمٌ }
قَتَلَ بَعْدَ قَبُولِ الدِّيَةِ
