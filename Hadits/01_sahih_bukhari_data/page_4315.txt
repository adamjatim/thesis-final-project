Ayat:
حَدَّثَنَا الْحُمَيْدِيُّ حَدَّثَنَا سُفْيَانُ حَدَّثَنَا عَمْرٌو قَالَ قَرَأَ ابْنُ عَبَّاسٍ
{ أَلَا إِنَّهُمْ يَثْنُونَ صُدُورَهُمْ لِيَسْتَخْفُوا مِنْهُ أَلَا حِينَ يَسْتَغْشُونَ ثِيَابَهُمْ }
و قَالَ غَيْرُهُ عَنْ ابْنِ عَبَّاسٍ
{ يَسْتَغْشُونَ }
يُغَطُّونَ رُءُوسَهُمْ
{ سِيءَ بِهِمْ }
سَاءَ ظَنُّهُ بِقَوْمِهِ
{ وَضَاقَ بِهِمْ }
بِأَضْيَافِهِ
{ بِقِطْعٍ مِنْ اللَّيْلِ }
بِسَوَادٍ وَقَالَ مُجَاهِدٌ
{ إِلَيْهِ أُنِيبُ }
أَرْجِعُ
