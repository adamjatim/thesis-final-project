Nombor hadis: (2958)

Ayat:
حَدَّثَنَا الْحَسَنُ بْنُ مُحَمَّدٍ الزَّعْفَرَانِيُّ حَدَّثَنَا الْحَجَّاجُ بْنُ مُحَمَّدٍ عَنْ ابْنِ جُرَيْجٍ أَخْبَرَنِي عَبْدُ الْكَرِيمِ سَمِعَ مِقْسَمًا مَوْلَى عَبْدِ اللَّهِ بْنِ الْحَارِثِ يُحَدِّثُ عَنْ ابْنِ عَبَّاسٍ أَنَّهُ قَالَ
{ لَا يَسْتَوِي الْقَاعِدُونَ مِنْ الْمُؤْمِنِينَ غَيْرُ أُولِي الضَّرَرِ }
عَنْ بَدْرٍ وَالْخَارِجُونَ إِلَى بَدْرٍ لَمَّا نَزَلَتْ غَزْوَةُ بَدْرٍ قَالَ عَبْدُ اللَّهِ بْنُ جَحْشِ وَابْنُ أُمِّ مَكْتُومٍ إِنَّا أَعْمَيَانِ يَا رَسُولَ اللَّهِ فَهَلْ لَنَا رُخْصَةٌ فَنَزَلَتْ
{ لَا يَسْتَوِي الْقَاعِدُونَ مِنْ الْمُؤْمِنِينَ غَيْرُ أُولِي الضَّرَرِ }
وَ
{ فَضَّلَ اللَّهُ الْمُجَاهِدِينَ }
{ عَلَى الْقَاعِدِينَ دَرَجَةً }
فَهَؤُلَاءِ الْقَاعِدُونَ غَيْرُ أُولِي الضَّرَرِ
{ وَفَضَّلَ اللَّهُ الْمُجَاهِدِينَ عَلَى الْقَاعِدِينَ أَجْرًا عَظِيمًا }
دَرَجَاتٍ مِنْهُ عَلَى الْقَاعِدِينَ مِنْ الْمُؤْمِنِينَ غَيْرِ أُولِي الضَّرَرِ
قَالَ أَبُو عِيسَى هَذَا حَدِيثٌ حَسَنٌ غَرِيبٌ مِنْ هَذَا الْوَجْهِ مِنْ حَدِيثِ ابْنِ عَبَّاسٍ وَمِقْسَمٌ يُقَالُ هُوَ مَوْلَى عَبْدِ اللَّهِ بْنِ الْحَارِثِ وَيُقَالُ هُوَ مَوْلَى ابْنِ عَبَّاسٍ وَكُنْيَتُهُ أَبُو الْقَاسِمِ
