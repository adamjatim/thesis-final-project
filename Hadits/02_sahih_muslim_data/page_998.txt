{ حَافِظُوا عَلَى الصَّلَوَاتِ وَالصَّلَاةِ الْوُسْطَى }
فَلَمَّا بَلَغْتُهَا آذَنْتُهَا
فَأَمْلَتْ عَلَيَّ

{ وَقُومُوا لِلَّهِ قَانِتِينَ }
